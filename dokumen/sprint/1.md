# 🏃‍♂️ Sprint 1: Foundation, Auth & Wallet Core
**Durasi:** 2 Minggu (14 Hari Kerja) | **Target Utama:** Fondasi database aman (Client & Server), autentikasi 5 langkah + OTP, JWT sesi, dan tampilan beranda/wallet aktif.

---

## 🎯 Tujuan Sprint

Sprint 1 fokus pada pembuatan infrastruktur dasar monorepo sehingga aplikasi Client (Flutter), Backend API (NestJS), dan Dashboard Admin (Next.js) dapat berjalan dan terhubung untuk fungsi autentikasi dan inisialisasi dompet CBDC.

1. **Client App (Flutter):** Menghubungkan UI dummy registrasi & login ke API asli, menyiapkan database lokal Drift + SQLCipher terenkripsi, dan menampilkan saldo riil dari lokal.
2. **Server (Backend NestJS):** Setup proyek NestJS, konfigurasi Prisma ORM dengan PostgreSQL, implementasi modul autentikasi (JWT, OTP, Argon2id hash), serta endpoint `GET /wallet/balance` & `GET /wallet/resolve`.
3. **Dashboard (Next.js):** Setup proyek Next.js 15, konfigurasi autentikasi Admin dengan sesi JWT, halaman utama (Overview Stat Cards), dan tabel daftar pengguna (`User List`).

## 📱 1. Client App (Flutter / Android)

### A. Lingkup Tugas & Deliverables
- [ ] **Database Lokal Terenkripsi (Drift + SQLCipher):**
  - Implementasi skema tabel dasar di `AppDatabase`: `Users`, `WalletBalances`, `Transactions`, `DeviceSessions`.
  - Integrasi `DatabaseService` dengan `flutter_secure_storage` untuk mengamankan AES-256 key di Android Keystore / TEE.
- [ ] **Modul Autentikasi (API Integrated):**
  - **Login (`login_page.dart`):** Hubungkan dengan `POST /api/v1/auth/login`. Simpan token JWT & buka sesi lokal.
  - **Registrasi Step 1 (`register_step1_page.dart`):** Validasi email & no ponsel real-time ke `POST /api/v1/auth/check-availability`.
  - **Registrasi Step 2 (`register_step2_page.dart`):** Request OTP via `POST /api/v1/auth/send-otp` dan verifikasi 6-digit OTP via `POST /api/v1/auth/verify-otp`.
  - **Registrasi Step 3 (`register_step3_page.dart`):** Cek ketersediaan username via `GET /api/v1/auth/check-username/:username`.
  - **Registrasi Step 4 (`register_step4_page.dart`):** Simpan foto kamera secara lokal (akan diunggah di Sprint 3/KYC).
  - **Registrasi Step 5 (`register_step5_page.dart`):** Hash PIN 6 digit secara lokal dengan Argon2id, generate Ed25519 keypair (`SimpleKeyPair`), simpan *private key* di TEE, dan kirim *public key* (`public_key_b64`) beserta semua data ke `POST /api/v1/auth/register`.
- [ ] **Modul Beranda & Wallet:**
  - **Beranda (`home_page.dart`):** Ganti `DummyWalletBalanceProvider` dengan *Riverpod StreamProvider* yang mendengarkan perubahan tabel `WalletBalances` di Drift lokal.

## ⚙️ 2. Server (Backend NestJS + Prisma)

### A. Lingkup Tugas & Deliverables
- [ ] **Inisialisasi Proyek & Database:**
  - Setup NestJS dengan TypeScript, konfigurasi Docker Compose untuk PostgreSQL 16 & Redis.
  - Implementasi `prisma/schema.prisma` berdasarkan `nirpay_backend_schema.dbml` (fokus tabel `users`, `wallet_balances`, `device_sessions`, `otp_verifications`, `password_resets`, `audit_logs`).
  - Jalankan `npx prisma migrate dev --name init_sprint1`.
- [ ] **Modul Autentikasi (`/api/v1/auth/*`):**
  - `POST /auth/check-availability` (email / no ponsel) & `GET /auth/check-username/:username`.
  - `POST /auth/send-otp` & `POST /auth/verify-otp` (simpan hash OTP ke tabel `otp_verifications`, expired 5 menit).
  - `POST /auth/register`: Validasi input (`class-validator`), hash password & PIN dengan Argon2id, simpan `public_key_b64` dari client, inisialisasi `wallet_balances` (saldo 0), generate JWT (Access 15m, Refresh 30d).
  - `POST /auth/login`: Verifikasi password Argon2id, catat `device_sessions`, kembalikan JWT & info user.
  - `POST /auth/refresh` & `POST /auth/logout`.
- [ ] **Modul Wallet (`/api/v1/wallet/*`):**
  - `GET /wallet/balance`: Kembalikan `amount_cent`, `reserved_cent`, `spendable_cent`, `max_hop`.
  - `GET /wallet/resolve/:identifier`: Cari user berdasarkan UUID server, email, atau username (untuk persiapan transfer).

### B. Spesifikasi Endpoint Utama

#### 1. `POST /api/v1/auth/register`
**Request Body:**
```json
{
  "email": "widya@example.com",
  "phone_number": "+6281234567890",
  "password": "Password123!",
  "full_name": "Widya Fitriadi",
  "username": "widyaf",
  "birth_date": "1999-05-15",
  "gender": "MALE",
  "pin": "123456",
  "public_key_b64": "MCowBQYDK2VwAyEAX... (Ed25519 Public Key dari Client)"
}
```
**Response (201 Created):**
```json
{
  "status": "success",
  "data": {
    "server_id": "8f9a2b3c-4d5e-6f7a-8b9c-0d1e2f3a4b5c",
    "access_token": "eyJhbGciOiJIUzI1Ni...",
    "refresh_token": "eyJhbGciOiJIUzI1Ni...",
    "user": {
      "email": "widya@example.com",
      "full_name": "Widya Fitriadi",
      "username": "widyaf",
      "kyc_status": "UNVERIFIED"
    }
  }
}
```
